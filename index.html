Introducción

La programación orientada a objetos (POO) es un paradigma fundamental en el desarrollo de software moderno. En esta práctica calificada se implementan tres ejercicios que demuestran los conceptos principales de POO:

Encapsulamiento

Ocultar los detalles internos de implementación

Abstracción

Representar características esenciales sin incluir detalles

Herencia

Mecanismo para crear nuevas clases basadas en clases existentes

Polimorfismo

Capacidad de objetos de diferentes clases de responder al mismo mensaje

Objetivos

Objetivo General

Implementar aplicaciones utilizando los principios de la Programación Orientada a Objetos en Java y Python usando el editor de código Google Colab.

Objetivos Específicos

  • Crear clases con atributos y métodos que modelan entidades del mundo real
  • Implementar constructores y métodos de acceso (getters/setters)
  • Aplicar el concepto de herencia entre clases
  • Desarrollar programas interactivos que soliciten datos por consola
  • Realizar cálculos y validaciones dentro de los métodos de las clases

Ejercicio 1: Clase Planeta

Puntuación: 6 puntos
Tecnología: Python + Tkinter

Descripción: Implementación de una clase que modela planetas del sistema solar con cálculos de densidad y determinación de si son planetas exteriores o interiores.

Código - Clase Planeta
import tkinter as tk
from tkinter import ttk, messagebox

class TipoPlaneta:
    def __init__(self, id_planeta: int, tipo_planeta: str):
        self.id_planeta = id_planeta
        self.tipo_planeta = tipo_planeta

    def __str__(self):
        return self.tipo_planeta

class Planeta:
    def __init__(self, nombre, satelites, masa, volumen, diametro, distancia_sol, tipo_planeta):
        self.nombre = nombre
        self.satelites = satelites
        self.masa = masa
        self.volumen = volumen
        self.diametro = diametro
        self.distancia_sol = distancia_sol
        self.tipo_planeta = tipo_planeta

    def calcular_densidad(self):
        if self.volumen == 0:
            return 0
        return self.masa / self.volumen

    def es_planeta_exterior(self):
        UA = self.distancia_sol * 1_000_000 / 149_597_870
        return UA > 3.4

    def mostrar_info(self):
        info = f"Nombre: {self.nombre}\nSatélites: {self.satelites}\nMasa: {self.masa:.2e} kg\n"
        info += f"Volumen: {self.volumen:.2e} km³\nDiámetro: {self.diametro} km\n"
        info += f"Distancia al Sol: {self.distancia_sol} millones de km\n"
        info += f"Tipo: {self.tipo_planeta}\nDensidad: {self.calcular_densidad():.2e} kg/km³\n"
        
        if self.es_planeta_exterior():
            info += "Este planeta es EXTERIOR al sistema solar.\n"
        else:
            info += "Este planeta es INTERIOR al sistema solar.\n"
        return info

Ejercicio 3: Clase Cuenta Bancaria (Herencia)

Puntuación: 8 puntos
Tecnología: Java

Descripción: Implementación de un sistema bancario con herencia, incluyendo cuentas de ahorros y cuentas corrientes.

Código - Sistema Bancario (Java)
// Clase base CuentaBancaria
public class CuentaBancaria {
    protected double saldo;
    protected double tasaInteresAnual;
    protected int transacciones;
    
    public CuentaBancaria(double saldoInicial, double tasaInteres) {
        this.saldo = saldoInicial;
        this.tasaInteresAnual = tasaInteres;
        this.transacciones = 0;
    }
    
    public void depositar(double monto) {
        if (monto > 0) {
            saldo += monto;
            transacciones++;
            System.out.println("✓ Depósito exitoso: $" + String.format("%.2f", monto));
        }
    }
    
    public void retirar(double monto) {
        if (monto > 0 && monto <= saldo) {
            saldo -= monto;
            transacciones++;
            System.out.println("✓ Retiro exitoso: $" + String.format("%.2f", monto));
        } else {
            System.out.println("✗ Error: Fondos insuficientes");
        }
    }
    
    public void calcularInteresMensual() {
        double interesMensual = saldo * (tasaInteresAnual / 100) / 12;
        saldo += interesMensual;
        System.out.println("✓ Interés mensual: $" + String.format("%.2f", interesMensual));
    }
    
    // Getters y otros métodos...
}

// Clase hija CuentaAhorros
public class CuentaAhorros extends CuentaBancaria {
    private boolean activa;
    
    public CuentaAhorros(double saldoInicial, double tasaInteres) {
        super(saldoInicial, tasaInteres);
        this.activa = saldoInicial >= 10000;
    }
    
    @Override
    public void retirar(double monto) {
        if (activa) {
            super.retirar(monto);
            // Verificar si la cuenta se desactiva después del retiro
            if (saldo < 10000) {
                activa = false;
                System.out.println("✗ Cuenta DESACTIVADA (Saldo < $10,000)");
            }
        } else {
            System.out.println("✗ Error: Cuenta inactiva. No se puede retirar.");
        }
    }
    
    @Override
    public void depositar(double monto) {
        super.depositar(monto);
        // Verificar si la cuenta se activa después del depósito
        if (!activa && saldo >= 10000) {
            activa = true;
            System.out.println("🎉 ¡Cuenta ACTIVADA! (Saldo ≥ $10,000)");
        }
    }
    
    // Otros métodos específicos de CuentaAhorros...
}
