Crear un Repositorio en Bitbucket

Guía paso a paso para crear tu primer repositorio y subir tu código

Proceso Completo

Iniciar sesión en Bitbucket

Accede a tu cuenta en bitbucket.org. Si no tienes una cuenta, crea una primero.

Crear un nuevo repositorio

Haz clic en el botón "Create" en el menú lateral y selecciona "Repository".

Nota: Asegúrate de estar en el workspace correcto donde quieres crear el repositorio.

Configurar el repositorio

Completa la información requerida:

  • Nombre del repositorio: composers (o el nombre que prefieras)
  • Descripción: Descripción opcional de tu proyecto
  • Workspace: group-composers
  • Control de acceso: Público o Privado

Agregar colaboradores

En la configuración del repositorio, agrega a los miembros de tu equipo con los permisos adecuados.

Importante: Asegúrate de asignar los permisos correctos (Lectura, Escritura o Administración).

Inicializar Git localmente

Abre tu terminal y navega a la carpeta de tu proyecto. Luego ejecuta:

git init
git add .
git commit -m "Initial commit"

Conectar con el repositorio remoto

Agrega el repositorio de Bitbucket como origen remoto:

git remote add origin https://bitbucket.org/group-composers/composers.git

Nota: Asegúrate de usar la URL correcta de tu repositorio.

Subir el código a Bitbucket

Ejecuta el comando para subir tu código al repositorio remoto:

git push -u origin main

¡Éxito! Tu código ahora está en Bitbucket y disponible para tu equipo.

Verificar en Bitbucket

Visita tu repositorio en Bitbucket para confirmar que todos los archivos se hayan subido correctamente.
